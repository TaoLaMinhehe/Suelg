# 🚀 Multi Tunnel Local Server Tool

> Simple Bash tool to run a local PHP server and expose it to the internet.

---

## ⚡ Features

* 🌐 Run local PHP server instantly
* 🔗 Multiple tunneling:

  * Ngrok
  * Cloudflared
  * LocalXpose
* 🎨 Colorful CLI interface
* ⚙️ Auto setup environment
* 📦 Works on Linux / Ubuntu / Termux

---

## ▶️ Usage

```bash
./script.sh
```
